# 雅思词库 · 刷词（PWA）

纯前端拼写练习，可安装到手机主屏幕，像 App 一样使用。  
**不需要自己的域名**，使用 GitHub Pages 免费地址即可。

## 线上地址

部署成功后为：

```text
https://lucas-736.github.io/ielts-vocab-trainer/
```

（若仓库名或用户名不同，以 GitHub Pages 设置为准。）

## 手机安装成「软件」

### iPhone

1. 用 **Safari** 打开上面的网址（不要用微信内置浏览器）
2. 点底部分享按钮
3. 选择 **添加到主屏幕**
4. 以后从桌面图标打开，全屏运行

### 安卓

1. 用 **Chrome** 打开网址
2. 菜单 → **安装应用** / **添加到主屏幕**  
   或点页面上的 **安装到桌面**
3. 从桌面图标打开

## 本地文件

```text
/Users/tom/Documents/雅思备考/02-词汇/vocab-trainer/
```

电脑也可双击 `index.html`（本地文件模式无法完整使用 PWA 安装与离线缓存）。

## 功能

- 拼写练习（中文 → 英文）
- 提示 / 读音 / 易错词 / 掌握标记
- 翻卡模式
- 词库列表与 JSON 导入导出
- PWA：可离线打开已缓存页面与词库

## 词库更新

改 `words.js` 后推送到 GitHub `main` 分支，网站会自动更新。  
进度存在各设备浏览器本地，**默认手机和电脑不同步**。

## 技术说明

| 文件 | 作用 |
|------|------|
| `index.html` | 界面与逻辑 |
| `words.js` | 词库 |
| `manifest.webmanifest` | 安装名称、图标、全屏 |
| `sw.js` | 离线缓存 |
| `icons/` | 应用图标 |
