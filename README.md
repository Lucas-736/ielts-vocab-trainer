## 页面结构（2026-07 重构）

**选什么词**：顶部一条选择条，两个维度说清在刷哪批词
- **章节 ▾**：全部章节 / Day 1 / Day 2 …（带词数）
- **范围 ▾**：全部 / 未认读 / 薄弱 / ★重点 / 核心6词 / 已掌握（带该章节下的命中数）
- **🔀** 打乱顺序（开关式），**↻** 重开一轮

**怎么练**：底部固定操作栏，主操作位置永不变化
- 认读教学 → `记住了，下一个`
- 认读自测 → `不认识` / `认识`（键盘 1 / 2）
- 拼写 → `检查`（回车）；`提示 · 读音 · 跳过` 弱化为文字按钮
- 「跳过」记一次拼错，所以是警示色

标签页只剩 **学习 / 词库 / 设置**；原「薄弱」页并入词库（范围选「薄弱」即按错误权重排序）。

## 外观与语言

- 右上角 **🌙 / ☀** 切换 **夜·墨 / 昼·纸** 两套主题；首次打开跟随系统深浅色，之后记住你的选择（`ielts_vocab_theme_v1`）。
- 右上角 **🌐** 切换 **中文 / English** 界面；词条释义仍是中文（那是学习内容）。语言存在 `ielts_vocab_lang_v1`。
- 「⚙ 设置」页顶部也有同样的两个开关。

## 技术要点（给 AI 的快捷参考）

| 点 | 说明 |
|----|------|
| 词库 | `words.js` 唯一权威数据源 |
| 进度 | `ielts_vocab_progress_v2` |
| 学习队列 | `buildQueue()` 在 `index.html` 内 |
| 薄弱分 | `weakScore()`：拼错权重更高 |
| 部署 | 推送到 `main` 分支，Pages 自动构建 |
| 缓存 | 改 `sw.js` 的 CACHE 版本号 +1 |

## 常见任务速查

| 任务 | 做什么 |
|------|--------|
